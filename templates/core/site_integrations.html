1

Criar Service Account no Google Cloud

  1. Acesse Google Cloud → Service Accounts
  2. Clique em + Criar Conta de Serviço
  3. Nome: "RankPulse" → clique Criar e Continuar
  4. Role: deixe vazio → Concluir
2

Baixar a chave JSON

Na lista de Service Accounts, clique na conta criada → aba Chaves → Adicionar chave → Criar nova chave → Tipo: JSON → Criar. O arquivo JSON será baixado automaticamente.

3

Conceder acesso

Para GA4: Em analytics.google.com → Admin → Acesso à propriedade → adicione o email da Service Account (ex: rankpulse@projeto.iam.gserviceaccount.com) com role Viewer.

Para GSC: Em search.google.com/search-console → Configurações → Usuários e permissões → adicione o mesmo email com Acesso total.

4

Colar o conteúdo JSON aqui

Abra o arquivo JSON baixado com qualquer editor de texto, selecione todo o conteúdo (Ctrl+A) e cole no campo abaixo. Você pode usar o mesmo JSON para GSC e GA4 se quiser.
